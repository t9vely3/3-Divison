    엑셀 / 구글 시트에서 데이터 가져오는 법
    방법 1 — 엑셀 파일 (.xlsx)
    엑셀 파일을 그대로 아래에서 선택하면 됩니다. 가장 안전한 방법 (한글 깨짐, 줄바꿈 문제 없음).

    방법 2 — 구글 시트 → 엑셀로 다운로드
    구글 시트 → 파일 → 다운로드 → Microsoft Excel(.xlsx) → 다운로드된 파일 선택

    방법 3 — CSV (인코딩 주의)
    구글 시트 → 파일 → 다운로드 → 쉼표로 구분된 값(.csv) → CSV 파일 선택
    💡 자동 처리: 생년월일 형식, 배정현황 JSON, 수강료 숫자, 한글 인코딩
    💡 같은 원서번호가 있으면 기존 데이터를 덮어씁니다 (업데이트 효과)
    엑셀(.xlsx) 또는 CSV 파일을 선택하세요
    또는 이 영역에 드래그 & 드롭 · .xlsx 권장
